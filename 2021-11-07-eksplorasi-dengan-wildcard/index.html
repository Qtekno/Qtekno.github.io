Eksplorasi dengan Wildcard

Publish date: 2021-11-07
Tags: wildcard

Berikut beberapa contoh eksplorasi yang bisa dilakukan pada wildcard di Linux. Memanfaatkan fitur wildcard membuat pekerjaan menjadi lebih efektif.

$ rm *
$ mv *linux .html dir1
$ less d*.txt
$ rm junk.???
$ ls hda[0-9]
$ ls hda[0-9][0-9]

Penjelasan perintah di atas:

  1. Perintah untuk menghapus seluruh file di direktori aktif. Hati-hati dalam menggunakan perintah ini, karena kesalahan menentukan direktori aktif akan berakibat fatal. File yang terhapus tidak dapat dikembalikan (undelete).
  2. Misalkan di working direktori terdapat file HTML dan ingin memindahkan semua file tersebut yang mengandung kata “linux” di dalamnya ke direktori tujuan bernama dir1.
  3. Untuk menampilkan semua file yang berawalan ’d' dan berakhiran ‘txt’.
  4. Menghapus semua file yang namanya berawalan dengan ‘junk’. dan diikuti tepat tiga karakter.
  5. Menampilkan semua file atau direktori yang namanya berawalan dengan ‘hda’, dan diikuti tepat satu angka.
  6. Menampilkan semua file atau direktori yang namanya berawalan dengan ‘hda’, dan diikuti tepat dua angka.

Bisa menjelaskan perintah ini?

$ ls {hd,sd}[a-c]
$ cp [A-Z]* dir2
$ rm *[!cehg]
