Perintah Melihat Hardware

Publish date: 2021-11-07
Tags: cli

Linux sebagai sistem operasi telah dilengkapi berbagai utilitas untuk melakukan tugas tertentu, salah satunya adalah utilitas untuk melihat system linux termasuk informasi hardaware di sistem Linux yang digunakan. Berikut ini beberapa perintah untuk menampilkan informasi hadware mesin (komputer) secara detail yang perlu kita ketahui.

Informasi tentang CPU

Untuk melihat spesifikasi cpu, kita dapat menggunakan perintah:

  1. cat /proc/cpuinfo
  2. lscpu
$ cat /etc/cpuinfo
vendor_id	: AuthenticAMD
cpu family	: 21
model		: 112
model name	: AMD A9-9425 RADEON R5, 5 COMPUTE CORES 2C+3G
stepping	: 0
microcode	: 0x6006704
cpu MHz		: 1906.148
cache size	: 1024 KB
...
...

Bisa juga menggunakan opsi grep untuk mem-filter keluaran:

$ cat /proc/cpuinfo | grep "model name"
model name	: AMD A9-9425 RADEON R5, 5 COMPUTE CORES 2C+3G
model name	: AMD A9-9425 RADEON R5, 5 COMPUTE CORES 2C+3G

Informasi Hardware

Untuk mendapatkan informasi hardware di sistem Linux, utilitas lshw (List Hardware) biasa digunakan. Utilitas ini akan meng-ekstrak informasi dari file /proc/. Disarankan sebgai super user dalam menjalankan perintah lshw. Berikut contoh penggunaan perintah lshw di komputer saya:

$ sudo lshw 
[sudo] password for wido:
ns                          
    description: Notebook
    product: 81D5 (LENOVO_MT_81D5_BU_idea_FM_ideapad 330-14AST)
    vendor: LENOVO
    version: Lenovo ideapad 330-14AST
    serial: PF1GFGHC
    width: 64 bits
    capabilities: smbios-3.0 dmi-3.0 smp vsyscall32
...
...

Dapat juga menambahkan opsi -short agar tampilan lebih singkat.

$ sudo lshw -short
[sudo] password for wido:
H/W path         Device      Class          Description
=======================================================
                             system         81D5 (LENOVO_MT_81D5_BU_idea_FM_ideapad 330-14AST)
/0                           bus            LNVNB161216
/0/21                        memory         160KiB L1 cache

/0/20                        processor      AMD A9-9425 RADEON R5, 5 COMPUTE CORES 2C+3G
/0/23                        memory         4GiB System Memory
/0/23/0                      memory         4GiB SODIMM DDR4 Synchronous Unbuffered
                                            (Unregistered) 2133 MHz (0,5 ns)
/0/0                         memory         128KiB BIOS

Informasi tentang Memori

Gunakan perintah “cat /proc/meminfo” untuk melihat ukuran memori yang terpasang di mesin dan perintah “free -m” untuk melihat memori yang tidak digunakan (free memori).

$ cat /proc/meminfo
MemTotal:        3474396 kB
MemFree:          372456 kB
MemAvailable:    1104976 kB
Buffers:          116556 kB
Cached:           837448 kB
SwapCached:         1284 kB

$ cat /proc/meminfo | head -2
MemTotal:        3474396 kB
MemFree:          380256 kB

$ free -m
       total       used        free      shared  buff/cache   available
Mem:   3392        2027         368          47         997        1084
Swap:  2718          10        2708
